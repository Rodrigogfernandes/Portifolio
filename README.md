# Portfólio de Rodrigo Guedes Fernandes

Bem-vindo ao meu portfólio pessoal! Este projeto é uma apresentação das minhas habilidades como desenvolvedor júnior, incluindo meus trabalhos, conhecimentos em linguagens de programação e informações pessoais.

## Descrição

Este portfólio é uma página web responsiva que exibe minhas informações profissionais, habilidades técnicas e pessoais, idiomas, educação, projetos desenvolvidos e uma seção de contato. Utiliza HTML, CSS e JavaScript para criar uma interface interativa com seções em acordeão.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e responsividade
- **JavaScript**: Interatividade (acordeão e formulário de contato)
- **Normalize.css**: Reset de estilos para consistência entre navegadores
- **Google Fonts**: Fonte Open Sans

## Funcionalidades

- **Seções Interativas**: Acordeão para organizar informações
- **Responsividade**: Design adaptável para dispositivos móveis e desktop
- **Formulário de Contato**: Campo para envio de mensagens (com validação básica)
- **Links Externos**: Conexões para WhatsApp, e-mail e repositórios no GitHub

## Como Executar

1. Clone o repositório:
   ```
   git clone https://github.com/Rodrigogfernandes/Portifolio.git
   ```

2. Navegue até a pasta do projeto:
   ```
   cd Portifolio
   ```

3. Abra o arquivo `index.html` em um navegador web.
